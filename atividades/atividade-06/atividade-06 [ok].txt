2. Atividade com Array Multidimensional (Matriz):
- Descri√ß√£o: Crie um programa que leia uma matriz 3x3 (ou seja, uma matriz com 3 linhas e 3 colunas) e exiba a soma dos elementos da diagonal principal.
- Dica: Use um loop `for` aninhado para ler os elementos da matriz e calcule a soma dos elementos da diagonal principal.

#include <stdio.h>

int main() {
    int matriz[3][3]; // matriz 3x3
    int soma_diagonal = 0; // vari√°vel para calcular a soma da diagonal principal

    // Loop aninhado para ler os elementos da matriz
    for (int i = 0; i < 3; i++) {

    }

    // Mostrar a matriz
    printf("Matriz:\n");
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            printf("%d ", matriz[i][j]);
        }
        printf("\n");
    }

    // Loop aninhado para calcular a soma da diagonal principal
    for (int i = 0; i < 3; i++) {
        soma_diagonal += matriz[i][i];
    }

    printf("A soma da diagonal principal √©: %d\n", soma_diagonal);
